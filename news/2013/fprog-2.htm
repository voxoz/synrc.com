Конференция kiev::fprog

Мы расскажем как использовать различные базы данных для Erlang: Mnesia, Riak, KAI; особенности использования секондари индексов, автоматическое построение REST интерфейсов к данным, использование Parse Transform для генерации JSON парсеров, обеспечение конкурентного доступа к данным, аспекты использования Feed Server и Feed UI.
