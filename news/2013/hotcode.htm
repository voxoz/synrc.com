Конференция Hotcode

Компания «Synrc Research Center» представит свой самый быстрый Erlang Web фреймворк N2O работающий на вебсокетах, а также проведет часовую Live Coding сессию по написанию LDAP сервера.

1 Июня — Презентация N2O

— Компьютер: Windows, Linux или Mac
— Почтовый клиент: Outlook Express aka Live Mail, Apple Mail, Claws Mail
— Web Browser с поддержкой WebSockets
— vi, emacs или FAR
— Erlang
— git
— rebar
— GitHub.com аккаунт
